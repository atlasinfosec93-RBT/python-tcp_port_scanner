simple tcp port scanner writen in python # Python TCP Port Scanner

A simple TCP port scanner written in Python for learning networking and cybersecurity concepts.

## Features

* Scan common TCP ports
* Scan custom TCP ports
* Lightweight and beginner-friendly
* Built using Python's `socket` module

## Requirements

* Python 3.x
* Git (optional)

## Installation

Clone the repository:

```bash
git clone https://github.com/atlasinfosec93-RBT/python-tcp_port_scanner.git
```

Move into the project directory:

```bash
cd python-tcp_port_scanner
```

Run the scanner:

```bash
python scanner.py
```

> On Linux or macOS, you may need to use:

```bash
python3 scanner.py
```

## Usage

1. Enter the target IP address.
2. Choose whether to scan common ports (`y`) or specify your own (`n`).
3. View the scan results.

## Disclaimer

This project is intended for educational purposes only. Only scan systems that you own or have explicit permission to test.
